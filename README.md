# 🔥 HabitForge — AI-Powered Habit Tracker

HabitForge helps you build consistent habits with streak tracking, a 35-day heatmap, and an AI coach that keeps you accountable — or roasts you when you slack off.

## Features

- 🔥 **Streak Tracking** — Daily streak counter for every habit you add
- 📊 **35-Day Heatmap** — GitHub-style contribution graph showing your consistency over time
- 🤖 **AI Coach** — Personalized motivational (or brutally honest) feedback based on your streak
- ✅ **One-Click Check-in** — Mark habits done for today in one tap
- 🎯 **Custom Habits** — Add any habit with a name and emoji
- 📈 **Dashboard Stats** — Combined streak, average completion rate, active habits count

## How It Works

1. Add a habit with a name and emoji
2. Mark it done every day to grow your streak
3. Hit **AI Coach** on any habit to get personalized feedback
4. Watch your heatmap fill up over time

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Vercel (Deployment)

## Run Locally

```bash
npm install
npm run dev
```
